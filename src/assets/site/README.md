# 사이트 이미지·영상 자산

`ImageSlot` 컴포넌트가 이 폴더를 자동으로 읽어 실제 자산으로 렌더한다.

## 규칙

- **파일 이름 = 슬롯 ID**. 예: `IMG-001.jpg`, `IMG-002.png`, `VID-003.mp4`
  - 확장자·대소문자 무관 (`.jpg/.jpeg/.png/.webp/.avif`, 영상 `.mp4/.webm`)
  - 파일이 있으면 그 슬롯이 실제 자산으로, 없으면 자리표시 패널로 렌더된다.
- 슬롯 ID ↔ 컷 내용 매핑: `docs/기획/image_requests.csv`
- Astro가 빌드 시 자동으로 WebP/AVIF 변환·반응형 생성한다(`astro:assets`). 원본은 고해상도 1장만 넣으면 된다.

## 금지 (CLAUDE.md)

- 사업장·작업·결과물·사람 사진의 AI 생성 금지. 스톡 사진을 실제 자산 자리에 사용 금지.
- 실제 촬영/보유 자산만 넣는다.
