# 00_핵심방향요약

> 기획서 전체의 기준이 되는 한 장 요약. 이후 모든 페이지·카피·이미지 지시는 이 방향을 따른다.

---

## 사업체 / 업종

- 상호: 모터리페어 (장용석의 내차를 부탁해 동탄신도시점)
- 업종: 수입차 전문정비 (BMW·벤츠·아우디·폭스바겐·미니·포르쉐)
- 위치·서비스 지역: 경기 화성 동탄 / **동탄·화성·오산·평택** 중심
- 연락: 전화 010-3203-7607 · 네이버 예약(스마트플레이스) · 블로그 blog.naver.com/extran

## 핵심 가치 (한 문장)

> 동탄에서 20년 경력 정비사가 브랜드 전용 진단기로 원인을 정확히 짚고, 작업 과정을 사진으로 투명하게 공유하는 수입차 전문 정비소.

## 카피 톤 판단 (근거 한 줄)

- 판단: **전문성 강조형 + 담담한 톤(혼합).**
- 근거: 고관여 업종이라 근거·수치로 신뢰를 만들어야 하고, 사장님이 원한 "고급스럽고 담담한" 결과 + 후기 반복 키워드("꼼꼼·자세히 설명")가 모두 차분한 전문가 어투를 가리킨다.

## 브랜드 보이스 요약

- 말투: 해요체 기반의 차분한 존댓말. 과장·감탄사 절제.
- 인칭: "모터리페어는…" / 고객 지칭: "고객님" · "차주분"
- 톤 키워드 3개: **정확한 · 꼼꼼한 · 투명한**
- 전문 용어는 쓰되 괄호로 풀고, 증상(고객 언어)을 앞세운다.

## 주 전환 목표

1. **전화 상담** (010-3203-7607, 탭 투 콜) — 확인형(증상·급함)·단골의 주 동선.
2. **네이버 예약**(스마트플레이스) — 비교형이 검토 후 예약하는 동선.
   - 두 CTA를 히어로·각 섹션 하단·플로팅에 상시 노출. 정적 사이트라 폼 백엔드가 없으므로 전환은 이 둘을 축으로 한다.

## 전체 방향

1. **3초 안에 "동탄 수입차 정비소"임을 전달**하고, 페르소나별 불안(비교형=보증·바가지 / 확인형=속도·정확성 / 단골=없음)을 각 동선에서 분리 해소한다.
2. **가장 강한 전환 무기는 "사진으로 원인을 보여주는 투명성"** — 바가지 우려·실력 검증·후기 부족을 동시에 해소한다. 신뢰 요소(20년 경력·전용 진단기·ISO 9001·14001·45001·A/S 1년·2만km)는 한 곳에 몰지 않고 페이지 전체에 분산 배치한다.
3. **블로그=정비 사례를 새 사이트에 네이티브로** 둔다(주간 1편 누적). 네이버 블로그는 신뢰 채널로 유지하되, 자체 도메인 사례 글이 사이트의 **SEO·GEO 자산**이 되어 "후기·사회적 증거가 얇다"는 약점을 메운다.
4. **GEO(AI 검색) 대응**: 각 사례·FAQ를 "한 문장만 떼어도 사실이 성립"하는 자기완결 형태로 작성. 정비 사례 표준 포맷(차종→증상→원인→수리→결과→사진), 장비 실명, 수치는 텍스트로.
5. **모바일 퍼스트 · 속도 우선.** 정적 사이트(Astro + Cloudflare Pages)로 빠르게, 이미지 자동 최적화(WebP·반응형·lazy), 전화/예약은 항상 한 손 안에.
6. **가격은 "작업 전 견적 안내"** 방식으로 통일하고, 가격 영역은 견적 문의 CTA로 대체한다.

## 기술·운영 전제 (이번 프로젝트에서 확정됨)

- 빌드: **Astro**(콘텐츠 컬렉션 + 이미지 자동 최적화) / 배포: **GitHub → Cloudflare Pages**
- 디자인: Apple식 토큰 시스템(단일 블루 액센트·SF Pro/Inter·풀블리드 타일). 단 **정비 사례 본문만 단일 리딩 컬럼**으로 분리(롱폼 포토에세이).
- 발행: 젬에서 글 → 사진 드롭 → Claude Code `publish-case` 스킬이 이미지 정리·alt/캡션·배치 자동화 → 검토 후 push.
