## 4. 의도적 제외 항목 (+이유)

| 항목 | 제외 이유 |
|---|---|
| 가격표·가격 범위 | "작업 전 견적 안내" 정책 — 가격 영역은 견적 CTA로 대체 (지침 확정) |
| 온라인 문의 폼 | 정적 사이트 폼 백엔드 없음 — 전화·네이버 예약 이원화. 향후 필요 시 별도 과제 |
| 픽업 서비스 | 미운영 확인(2026-07-08) — 언급하지 않음 |
| 리뷰 수(7건) 표기 | 거짓 없이 약점 강조 없이 — 평점만 표기, 사례 누적으로 대체 |
| 대표 인사말 별도 페이지 | 담담한 브랜드 톤과 충돌 — 진단장비 §4 한 섹션 + 사례 문체로 대체 |
| 네이버 블로그 글 마이그레이션 | 새로 시작 확정 — 빈 상태 기간만 블로그 링크로 연결 |
| 충주·제천·원주·음성 지역 | 잘못된 정보 — 전면 금지 (05 §2) |
