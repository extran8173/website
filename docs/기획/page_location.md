## 페이지 목적

Q12("어디에 있나 · 동탄/오산/평택에서 가까운가")에 즉답하고, 방문 직전 확인 사항(영업시간·주차·연락)을 한 화면에 준다. 짧고 사실 위주 — 이 페이지에 온 방문자는 이미 마음을 정한 상태이므로 설득이 아니라 마찰 제거가 목표다.

## SEO / 메타

- 타깃 키워드: **동탄 · 오산 · 평택 수입차 정비** (지역 조합)
- 타이틀: `오시는 길 · 영업시간 | 동탄 모터리페어`
- 메타 디스크립션: `경기 화성시 효행로1156번길 9. 동탄·오산·평택에서 가까운 수입차 정비소. 월–금 08:30–18:30, 토 08:30–15:00. 대차 2대 운용.`
- 로컬 SEO 핵심: 이 페이지의 상호·주소·전화·영업시간(NAP)은 네이버 플레이스·구글 비즈니스와 **글자 단위로 일치**시킨다 (03 공통 §6 단일 소스 규칙).

## 지도 처리 (정적 사이트 제약)

- 동적 지도 API 대신 **정적 지도 이미지 + 외부 링크** 방식을 기본으로 한다: 빠르고(모바일 속도 원칙), API 키·과금 관리가 없다.
- 네이버 지도·카카오맵 링크는 앱 딥링크로 열리므로 모바일에서 길찾기 UX가 오히려 더 좋다.
- 방문자 대부분이 한국인이므로 구글맵 링크는 생략 가능(원하면 추가).

## 확인 필요 (빌드 전 사장님 확인)

1. **지역별 실제 소요 시간** — §3의 10~15분/15분은 자리값. 실제 체감 시간으로 보정.
2. **주차 안내** — 정비 베이 앞 주차 가능 여부, 대기 주차 공간 유무. (첫 방문자 마찰 요소)
3. **네이버 플레이스에 등록된 영업시간**이 위 값과 일치하는지 — 불일치 시 플레이스 쪽을 수정.
