## 페이지 목적

"내 증상·내 차가 여기 대상인가"(B5)에 답하고, 각 정비 항목의 불안(가격·품질·정직성)을 항목 안에서 해소한 뒤 전화 상담으로 넘긴다. 확인형 페르소나(증상 발생·급함)의 핵심 경유지 — 증상으로 들어와 전화로 나가는 페이지.

### 1. 페이지 헤드

- H1: `증상만 말씀하셔도 됩니다` (display-lg)
- 서브: `전용 진단기로 원인을 확인하고, 견적을 안내한 뒤 승인받고 작업합니다.`
- 이미지 없음 — 텍스트 헤드. (해소: B2 선제)

### 2. 진행 방식 3단계 — 모든 항목 공통 프로세스

- H2: `모든 정비는 같은 순서로 진행합니다`
- 3단계 카드(텍스트+숫자, 아이콘 최소):

| 단계 | 카피 |
|---|---|
| 1. 진단 | 전용 진단기와 계측 장비로 원인을 먼저 확인합니다. |
| 2. 견적 안내 | 교환이 필요한 부위를 사진으로 보여드리고, 작업 전에 견적을 안내합니다. 승인 후에 작업합니다. |
| 3. 정비·기록 | 정비 과정을 사진으로 기록하고, 출고 시 함께 설명해 드립니다. |

- 하단 caption: `교환 부품 A/S: 차량 제작사 기준과 동일 (1년 / 2만km)`
- 해소: Q5(사전 견적), Q10(불필요 수리), B2(바가지). — 이 섹션이 서비스 페이지의 신뢰 축. 가격표가 없는 사이트에서 "작업 전 견적 + 사진 근거 + 승인 후 작업"이 가격 불안의 대체 답변이다.

### 3. 진단 — #diagnosis

- H2: `경고등이 떴어요 · 원인을 모르겠어요`
- 본문: `계기판 경고등, 시동 불량, 원인 모를 이상 증상은 추측으로 고치면 비용만 늘어납니다. 모터리페어는 BMW ISTA+, 벤츠 Xentry, 아우디 ODIS, 포르쉐 PIWIS로 차량 데이터를 직접 읽고, 필요하면 Picoscope(전기 신호 계측기)로 센서·배선 신호까지 확인해 원인을 짚습니다.`
- 근거 caption: `브랜드 전용 진단기 보유 — 진단 장비 페이지에서 실물을 확인하세요 →`
- 해소: Q3, Q7, B3. / 이미지: **IMG-004** PIWIS 화면 (보유). alt="포르쉐 PIWIS 전용 진단 화면"

### 4. 오일류 — #oil

- H2: `오일 교환 시기가 됐어요`
- 본문: `엔진오일·미션오일·브레이크액을 차종별 제조사 규격에 맞춰 교환합니다. 수입차는 규격에 맞지 않는 오일이 고장의 원인이 되기도 합니다. 교환 전 상태를 확인해 사진으로 보여드리고, 당일 정비로 진행됩니다.`
- 근거 caption: `오일류·소모품 정비는 당일 가능합니다.`
- 해소: Q4·Q6(품질), Q13(당일). / 이미지 없음 — 텍스트 중심(적합 컷 미보유). 향후 작업 컷 확보 시 추가.

### 5. 엔진·미션 — #powertrain (다크 타일)

- H2: `큰 수리, 사설에서 될까 고민되시죠`
- 본문: `엔진·미션 정비는 분해 경험이 결과를 좌우합니다. 대표는 수입차 서비스센터와 엔진 전문회사에서 20년 넘게 정비해 왔고, 진단부터 분해·조립까지 직접 진행합니다. 작업 과정은 단계별로 사진에 남겨 정비 사례로 공개합니다.`
- 근거 caption: `교환 부품 A/S 1년/2만km — 중정비도 동일하게 적용됩니다.`
- 해소: Q2, B1·B3. / 이미지: **IMG-005** 작업 베이 내부(리프트·엔진룸) 가로형 (보유). alt="동탄 수입차 정비소 엔진 정비 작업"

### 6. 브레이크·서스펜션 — #brake-suspension

- H2: `주행 중 소음이 나요 · 승차감이 이상해요`
- 본문: `소음과 승차감 문제는 원인 부위를 특정하는 게 먼저입니다. 리프트에 올려 하부를 점검하고, 마모 상태를 사진으로 보여드립니다. 브레이크 패드처럼 잔량이 남은 부품은 남은 수치를 그대로 알려드립니다 — 아직 쓸 수 있는 부품을 교환하자고 권하지 않습니다.`
- 해소: Q11(정직성), B2. / 이미지 없음 — 텍스트 중심. (하부 점검·모니터 설명 컷은 정비 사례 쪽에 배정)

### 7. 냉·난방 — #hvac

- H2: `에어컨이 안 나와요 · 냉각수가 부족하다고 떠요`
- 본문: `에어컨 성능 저하, 냉각수 감소 경고는 여름철 가장 잦은 입고 증상입니다. 냉매 압력과 누설 부위를 계측 장비로 확인하고, 냉각 계통은 압력 테스트로 새는 지점을 찾은 뒤 수리합니다. 임시 보충이 아니라 원인 부위를 잡는 게 목표입니다.`
- 해소: Q3(계절 증상 대응, "한 번에 잡아줄까"). / 이미지 없음 — 텍스트 중심.

### 8. 최근 정비 사례 — recent-cases

- H2: `실제로 이렇게 진행됩니다`
- case-card 3개 (공통 컴포넌트). — 항목별 설명의 증거 역할.

### 9. 최종 CTA — cta-section

```
  ├─ 문구: 어떤 증상이든, 전화로 먼저 말씀해 주세요.
  ├─ [전화로 증상 상담하기] [네이버로 예약하기]
  └─ caption: 대차 2대 운용 — 예약 시 말씀해 주세요.
```
- 대차를 여기 배치: 서비스(특히 중정비) 검토자의 마지막 망설임("차 없이 어떻게 다니지")을 전환 직전에 해소.
