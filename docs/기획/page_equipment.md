## 페이지 목적

비교형 페르소나("딜러 말고 사설에 맡겨도 괜찮나")의 실력 검증 페이지. 말이 아니라 **물증**으로 답한다 — 장비 실물 사진, 브랜드별 진단기 실명, 20년 경력, ISO 3종. B3(실력 검증)을 이 페이지 하나로 끝내고 정비 사례 또는 예약으로 넘긴다. 보유 사진이 가장 강한 페이지이므로 사진을 전면에 세운다.

## SEO / 메타

- 타깃 키워드: **수입차 진단기 정비소** (보조: ISTA 사설 정비소)
- 타이틀: `브랜드 전용 진단기 · ISO 인증 | 동탄 모터리페어`
- 메타 디스크립션: `BMW ISTA+, 벤츠 Xentry, 아우디 ODIS, 포르쉐 PIWIS, Picoscope 보유. 20년 경력 대표가 직접 진단하는 동탄 수입차 정비소. ISO 9001·14001·45001 취득.`
- GEO 대응: 장비별 설명을 "모터리페어는 ~을 보유하고 ~에 사용합니다" 자기완결 문장으로 작성 — "ISTA+ 쓰는 사설 정비소" 류 AI 질의의 인용 대상.

## 스크롤 구조도


### 1. 페이지 헤드 — 다크 타일

- 배경: **IMG-007** 진단 노트북 세트 ②(다른 각도, 어두운 톤과 어울리는 컷) 풀 와이드, 상단 스크림. alt="" (배경 장식용 — 실물은 §2에서 개별 노출)
- H1: `장비가 진단의 절반입니다` (hero-display, 흰색)
  - 메인 티저(§page_메인 5)와 동일 문장 — 티저에서 넘어온 방문자에게 연속성을 준다.
- 서브: `공식 서비스센터가 쓰는 진단기를 그대로 갖췄습니다.`

### 2. 브랜드별 진단기 — 장비 카드 4개

- H2: `브랜드별 전용 진단기`
- 도입 1줄: `수입차는 브랜드마다 진단 규격이 다릅니다. 범용 스캐너로는 읽지 못하는 데이터를 전용 진단기가 읽습니다.`
- 카드 4개 (store-utility-card 규격, 모바일 1단/데스크톱 2단). 각 카드: 브랜드명(body-strong) + 장비명 + 자기완결 설명 2줄:

| 브랜드 | 장비 | 설명 (자기완결형) |
|---|---|---|
| BMW · 미니 | ISTA+ | 모터리페어는 BMW 공식 진단 시스템 ISTA+를 보유하고, BMW·미니의 고장 코드 판독과 코딩·프로그래밍에 사용합니다. |
| 벤츠 | Xentry | 모터리페어는 벤츠 공식 진단 시스템 Xentry를 보유하고, 벤츠 전 차종의 제어기 진단과 기능 설정에 사용합니다. |
| 아우디 · 폭스바겐 | ODIS | 모터리페어는 아우디·폭스바겐 공식 진단 시스템 ODIS를 보유하고, 두 브랜드의 고장 진단과 부품 적응 작업에 사용합니다. |
| 포르쉐 | PIWIS | 모터리페어는 포르쉐 공식 진단 시스템 PIWIS를 보유하고, 포르쉐 전 차종의 진단과 정비 이력 관리에 사용합니다. |

- 카드 하단 공통 이미지: **IMG-008** 진단 노트북 세트 ①(전체 세트 컷, 가로형). alt="수입차 브랜드별 전용 진단기 노트북 세트"
- **IMG-009** PIWIS 화면(포르쉐 카드 내 개별 이미지, 정방형 크롭). alt="포르쉐 PIWIS 전용 진단 화면"
- 해소: Q1(브랜드 매칭 — 어떤 브랜드가 어떤 장비로 다뤄지는지), Q7, B3.

### 3. 정밀 계측 — Picoscope

- H2: `진단기가 못 찾으면, 신호를 직접 봅니다`
- 본문: `고장 코드만으로 원인이 좁혀지지 않는 경우가 있습니다. 그럴 때는 Picoscope(오실로스코프 — 전기 신호를 파형으로 보는 계측기)를 센서와 배선에 직접 연결해 신호를 확인합니다. 추측 대신 계측으로 판단합니다.`
- 이미지: **IMG-010** Picoscope가 엔진룸에 연결된 컷 (보유, 가로형). alt="Picoscope 정밀 진단 장비 엔진룸 계측"
- 해소: Q10(불필요 수리 — "추측 대신 계측"), B3 심화.

### 4. 장비를 다루는 사람 — 다크 타일

- H2: `장비는 도구입니다. 판단은 경력이 합니다` (흰색)
- 본문: `대표 장용석은 수입차 서비스센터와 엔진 전문회사에서 20년 넘게 정비해 왔습니다. 진단 데이터를 읽는 것과 원인을 판단하는 것은 다른 일입니다. 모든 차량의 진단과 판단을 대표가 직접 합니다.`
- 이미지: **IMG-011** 모니터 설명 컷(대형 모니터에 차량 하부 사진을 띄우고 설명하는 장면, 가로형 — 보유). alt="동탄 수입차 정비소 차량 상태 사진 설명"
  - 이 컷이 현재 보유분 중 "사람"이 나오는 유일한 컷 — 이 섹션에 배정한다. **대표 프로필 사진 확보 시 이 섹션에 추가**(이미지 CSV 확보 권장 등재).
- 해소: Q8, B3. — 장비 페이지에 사람 섹션을 두는 이유: "장비만 있고 못 다루는 곳 아닌가" 하는 2차 의심 차단.

### 5. 객관적 근거 — ISO 인증

- H2: `숫자로 확인할 수 있는 것들`
- 본문: `모터리페어는 ISO 9001(품질경영) · ISO 14001(환경경영) · ISO 45001(안전보건경영) 인증을 취득했습니다. 교환한 부품은 차량 제작사 기준과 동일하게 1년 / 2만km 동안 A/S를 적용합니다.`
- 이미지: **IMG-012** ISO 인증서 3종 (미보유 — **재확보 권장**, 인증서 촬영/스캔). 확보 전까지 텍스트 배지 3개(trust-badge-row 변형)로 대체. alt(확보 시)="모터리페어 ISO 9001 14001 45001 인증서"
- 해소: Q15, B1, 차별화 가치 5(객관적 신뢰 근거).

### 6. 최근 정비 사례 — recent-cases

- H2: `이 장비들로 이렇게 진단했습니다`
- case-card 3개. — 장비 → 실전 증거로 연결. 비교형 동선(장비 → 사례)의 다리.
