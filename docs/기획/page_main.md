## 페이지 목적

3초 안에 "동탄 수입차 정비소"임을 전달하고, 세 페르소나를 각자의 동선으로 분기시킨다 — 확인형은 즉시 전화, 비교형은 진단 장비·정비 사례로, 단골은 연락처로. 주 전환(전화·네이버 예약)이 화면 어디서든 한 번의 탭 거리에 있게 한다.

## SEO / 메타

- 타깃 키워드: **동탄 수입차 정비** (보조: 화성 수입차 정비)
- 타이틀: `동탄 수입차 전문 정비 | 모터리페어`
- 메타 디스크립션: `동탄·화성·오산·평택 수입차 정비. BMW·벤츠·아우디·포르쉐 전용 진단기로 원인을 진단하고, 작업 과정을 사진으로 보여드립니다. 부품 A/S 1년/2만km.` (공백 포함 약 100자)
- H1은 히어로 1개만. GEO 엔티티 선언 문장은 소개 블록 첫 문단에 배치(아래 §섹션 3).

[2 신뢰 배지] — trust-badge-row (공통 컴포넌트)

### 1. 히어로 — 풀블리드

- 배경: **IMG-001** 사업장 정면(간판) 사진, 풀 와이드. 밝은 낮, 간판·정비 베이 3칸이 보이는 컷. alt="동탄 수입차 정비소 모터리페어 외관"
- 텍스트 가독을 위해 하단 40% 검정 단색 스크림(그라디언트 장식 아님, 기능적 오버레이).

```
  ├─ H1: 동탄 수입차 전문 정비            (hero-display, 흰색, 중앙)
  ├─ 서브: 원인을 진단하고, 과정을 사진으로 보여드립니다.   (lead, 흰색)
  └─ CTA: [전화로 증상 상담하기](채움) [네이버로 예약하기](고스트)
```

- H1 12자 — 3초 전달 원칙. 서브는 차별화 1·2순위(진단력·투명성)를 한 줄에 압축.
- 해소: Q12(지역) 즉답. / 모바일에서 CTA 2개는 세로 스택.

### 2. 신뢰 배지 — trust-badge-row

```
  [경력 20년] [ISO 9001 · 14001 · 45001] [부품 A/S 1년 / 2만km]
```

### 3. 소개 한 문단 — parchment

- H2: `모터리페어는 이런 곳입니다` (display-md)
- 본문 (GEO 엔티티 선언 — 첫 문장은 떼어내도 사실이 성립하는 형태):

> 모터리페어는 경기 화성 동탄에서 BMW·벤츠·아우디·폭스바겐·미니·포르쉐를 정비하는 수입차 전문 정비소입니다. ISTA+·Xentry·ODIS·PIWIS 등 브랜드 전용 진단기로 고장 원인을 진단하고, 점검부터 출고까지 모든 과정을 사진으로 기록해 보여드립니다. 서비스센터와 엔진 전문회사에서 20년 넘게 정비해 온 대표가 직접 진단합니다.

- 해소: Q1(브랜드), Q7(장비), Q9(사진) 선언. / 이미지 없음 — 텍스트 중심.

### 4. 증상 진입 블록 — canvas

- H2: `이런 증상이신가요?`
- 증상 카드 5개 (모바일 1단 / 데스크톱 3+2단). 각 카드: 증상 문장(body-strong) + 연결 서비스 한 줄(caption) + 서비스 페이지 앵커 링크. 텍스트 카드 + 최소 아이콘 — 사진 불필요 섹션.

| 증상 (고객 언어 그대로) | 연결 |
|---|---|
| "경고등이 떴어요" | → 서비스 #진단 |
| "주행 중 소음이 나요" | → 서비스 #브레이크·서스펜션 |
| "에어컨이 안 나와요" | → 서비스 #냉난방 |
| "냉각수가 부족하다고 떠요" | → 서비스 #냉난방 |
| "오일 교환 시기가 됐어요" | → 서비스 #오일류 |

- 마지막에 텍스트 링크 1줄: "다른 증상도 전화로 말씀해 주세요 →" (tel:)
- 해소: Q3(증상 대응), B5(내 증상이 대상인지). 확인형 페르소나의 주 진입점.

### 5. 진단 장비 티저 — 다크 타일 (surface-tile-1)

- H2: `장비가 진단의 절반입니다` (흰색)
- 본문 2줄: `BMW ISTA+, 벤츠 Xentry, 아우디 ODIS, 포르쉐 PIWIS. 브랜드가 쓰는 진단기를 그대로 갖추고, Picoscope로 전기 신호까지 계측합니다.`
- 이미지: **IMG-002** 진단 노트북 세트(가로형, 다크 배경과 어울리는 컷). alt="수입차 브랜드 전용 진단기 ISTA Xentry ODIS PIWIS"
- CTA(고스트, on-dark): "어떤 장비로 진단하는지 보기 →" (진단 장비 페이지)
- 해소: Q7(장비), B3(실력 검증). 비교형 페르소나의 주 분기점.

### 6. 최근 정비 사례 — recent-cases

- H2: `최근 정비 기록`
- 서브 1줄: `모든 정비는 차종·증상·원인·수리·결과를 사진과 함께 기록합니다.`
- case-card 3개, Astro 컬렉션 최신순 자동. (발행 전 초기 상태: 준비 중 문구 + 네이버 블로그 링크로 임시 대체 — 첫 사례 발행 시 자동 전환)
- CTA: "정비 사례 전체 보기 →"
- 해소: Q8·Q9·Q10, B2·B4. 이 섹션이 메인의 신뢰 엔진.

### 7. 후기 — review-strip, parchment

- H2: `차주분들이 남긴 말`
- review-card 3개 — 구글 리뷰 캡처(**IMG-003**, 선명분)에서 발췌 인용 + 출처 표기:
  - "간단한 작업이라도 하나하나 꼼꼼히 체크해 주셔서 항상 믿고 차를 맡길 수 있어요." — 구글 리뷰
  - "직접 진단에 신경을 많이 쓰는 곳이에요. 자세히 설명해 주셔서 좋았습니다." — 구글 리뷰, BMW 320d 차주
  - "정비소 많이 다녀봤는데 진짜 무언가 다른 곳입니다." — 구글 리뷰
- 상단에 "구글 평점 ★4.7" 텍스트. 리뷰 수(7건)는 표기하지 않는다 — 거짓 없이, 강조도 없이.
- 해소: Q8, B4. ※ 인용은 캡처 원문 기준으로 빌드 시 재확인.

### 8. FAQ 3문항 — canvas

자연어 질문 그대로 (GEO). 아코디언 없이 펼쳐진 상태로 노출(AI 크롤러가 읽도록).

- **Q. 사설 정비소에서 수리하면 제조사 보증이 깨지나요?** (Q16/B1)
  A. 보증 기간 중인 차량은 입고 시 미리 알려주세요. 보증 수리 대상인 항목은 공식 서비스센터 이용이 유리할 수 있어 사전에 안내해 드리며, 모터리페어에서 교환한 부품은 제조사 기준과 동일하게 1년/2만km A/S를 적용합니다.
- **Q. 당일 정비도 가능한가요?** (Q13)
  A. 엔진오일 등 오일류와 소모품 교환 같은 메인터넌스 정비는 당일 가능합니다. 진단이 필요한 증상이나 중정비는 전화로 증상을 말씀해 주시면 소요 일정을 미리 안내해 드립니다.
- **Q. 수리하는 동안 차가 없으면 어떻게 하나요?**
  A. 대차 2대를 운용하고 있습니다. 예약 시 말씀해 주시면 일정에 맞춰 준비해 드립니다.

- 전체 FAQ는 오시는 길 페이지(구 문의, `/location/`)에 — 여기선 전환 직전 불안 3개만.
